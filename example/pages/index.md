# Welcome to Lence v2

Lence is a lightweight data visualization framework. Write SQL queries and visualize data with charts and tables.

## Getting Started

1. Define your data sources in `config/sources.yaml`
2. Create pages in the `pages/` directory
3. Use `{% query %}` tags to define SQL queries
4. Embed charts and tables with `{% chart %}` and `{% table %}` tags

## Features

- **SQL Queries** - Query CSV files with DuckDB
- **Charts** - Line, bar, pie, and more with ECharts
- **Tables** - Sortable data tables
- **Markdoc** - Write narrative content with Evidence.dev-style syntax
